# Madara Starknet Client

Madara is an open source stack that allows you to build appchains powered by Cairo and Starknet technology.

## It has 3 main Functionalities

- Appchain Stack: Allows to easily deploy your own Starknet instance to host you application
- Full node that allows to interact with diffrent Starknet networks
- Devnet: allows to simulate a Starknet network to test out you Cairo Smart Contracts
