# Starknet.js 概览

[Starknet.js](https://www.starknetjs.com/) 是一个轻量级的 JavaScript/TypeScript 库，能够使您的 DApp 与 Starknet 进行交互。借助 Starknet.js，您可以与 Starknet 的帐户、提供者和合约交互。

## 安装 Starknet.js

按照 Starknet.js 官方安装说明安装：https://www.starknetjs.com/docs/guides/intro

## 配置 Starknet.js 以适用于 Madara

要连接到正在运行的 Madara 节点，需要将提供者指向 Madara RPC URL。

```javascript
const provider = new starknet.RpcProvider({
  nodeUrl: "http://localhost:9944",
});
```

您现在可以按照 Starknet.js 的[文档](https://www.starknetjs.com/docs/guides/intro)中的说明，使用此提供者与链进行交互。

[Karnot](https://karnot.xyz) 还开发了一些使用 Starknet.js 的现成脚本，这些脚本可用于为钱包充值、声明和部署合约以及其他一些有用的任务。您可以在[这里](https://github.com/karnotxyz/madara-get-started)参考这些脚本。
