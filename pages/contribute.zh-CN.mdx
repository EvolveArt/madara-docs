# 为 Madara 贡献

Madara 自创立之日起，始终以尽可能开放源代码的方式构建，始终遵循推动 Starknet 生态系统的原则。任何人都可以自由加入 [Telegram 群组](https://t.me/MadaraStarknet)聊天、寻求帮助、讨论任何事情！

我们还定期组织社区会议，议程可以在 [GitHub 讨论](https://github.com/keep-starknet-strange/madara/discussions)中找到。

<Callout type="info" emoji="🧪">
  如果你有雄心勃勃的黑客创意，与堆栈的某一层相关，比如新的 DA 模式或新颖的 VM，那就大胆前往 GitHub 仓库创建帖子开始讨论吧。如果一切顺利，你将受邀在社区会议中展示你的想法，然后开始实施！
</Callout>

## 开发者

通过 [OnlyDust](https://app.onlydust.xyz/) 为 Madara 贡献即可获得奖励。<br></br>
_这是什么意思？_ 这意味着你作为一名开发者可以自由选择任何未解决的问题，并提交一个 PR（Pull Request）。一旦 PR 得到合并，你将根据 PR 的时间/难度获得相应的报酬。我们的贡献者已经超过 20 名，现在就等你加入我们 😎

## 寻找错误

发现了一个错误？请在 GitHub 上提交 [Issue](https://github.com/keep-starknet-strange/madara/issues)，这样任何开源贡献者都可以查看并修复（你也可以！）。

## 文档

在这些文档中遗漏了任何内容？发现了打字错误？请在 [GitHub](https://github.com/EvolveArt/madara-docs/) 提交 PR（拉取请求）或创建一个问题。正如我们所做的一切，我们共同成长，越多的人检查，就会变得越好。
