# Madara 证明器 API 概述

[Madara 证明器 API](https://github.com/Moonsong-Labs/madara-prover-api) 是一个全面的 RPC 服务器和客户端，旨在方便从 Madara 排序器执行 Stone 证明器。该项目依赖于强大的 [Stone 证明器](https://github.com/starkware-libs/stone-prover)，在证明每个 Madara 区块内的交易中发挥着至关重要的作用，。

## 说明

Madara Prover API 项目由三个主要的 Crate 组成：

1. **stone-prover**：这个 Crate 提供了 Stone 证明器的 Rust 封装。使用 `build.rs` 脚本来构建证明器，并提供了一个 Rust 库，可以无缝地后台作为子进程调用证明器。

2. **madara-prover-rpc-server**：这个 Crate 包含了服务器代码，提供了一个二进制文件和库来启动服务器。服务器负责在 Starknet OS 上运行任何一组 Cairo 程序，生成程序的状态差异和执行证明。

3. **madara-prover-rpc-client**：这个 Crate 提供了一个客户端，用于与服务器进行交互，通过 [gRPC](https://grpc.io/) 实现通信。协议描述可以在 `protocols/prover.proto` 中找到。
